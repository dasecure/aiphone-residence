How Aiphone IXG + PassQR
work together
The IXG hardware handles physical access — video intercom, QR scanning, door release. PassQR handles digital identity — wallet passes, validation logic, scan history, and AI management.

Data flow
End-to-End Data Flow
1
Pass issued & added to wallet
Management issues a pass via the portal, AI chat, or direct API. PassQR generates a signed QR code and pushes it to the resident's Apple or Google Wallet silently over the air. For visitors, a WhatsApp link is sent directly to their phone.
POST /aiphone-book → PassQR API → Wallet OTA push
2
Resident presents QR at IXG door station
The resident opens their Wallet (or the WhatsApp pass link) and holds the QR to the Aiphone IXG door station's scanner module. This works entirely offline on the resident's phone — no app, no internet required at the device.
Aiphone IXG QR scanner module → HTTP client
3
IXG posts QR code to PassQR validation endpoint
The IXG door station sends an HTTPS POST request containing the scanned QR code to the aiphone-scan edge function. This is the core integration point — configured once in the IXG's "external access control URL" setting.
POST https://...supabase.co/functions/v1/aiphone-scan
4
Validation, logging & door signal
The edge function validates the pass against PassQR, detects the pass type (resident/visitor/contractor/facility), increments the scan count, and logs the scan to Supabase with timestamp, gate, and holder details. It returns a JSON response that IXG maps to a door open or deny signal.
200 OK {valid:true} → IXG releases door · 4xx → access denied
5
Scan appears on live dashboard
The management dashboard polls Supabase every 4 seconds. The new scan entry slides into the live feed with the resident's name, unit, pass type, and timestamp. The "Active Now" panel shows everyone who has scanned in the last 2 hours.
aiphone-stats → Supabase DB → dashboard poll 4s
6
AI concierge provides natural language oversight
Property managers can ask the AI to look up any resident, create facility bookings, issue visitor passes, or revoke contractor access — all through plain English. Claude calls the PassQR MCP tools directly, with every action audited in Supabase.
AI chat → aiphone-ai → Claude Sonnet → PassQR MCP tools
